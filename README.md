
# 🔐 Login / Sign Up UI

A modern and responsive **Login / Sign Up interface** built using HTML, CSS, and JavaScript.
This project focuses on clean design, smooth user experience, and simple client-side authentication using LocalStorage.

---

## ✨ Features

* 🔄 Smooth switch between **Sign In** and **Sign Up**
* 🎨 Clean and modern **Minimal UI design**
* 📱 Fully **Responsive** (Mobile, Tablet, Desktop)
* ⚡ Lightweight animations for better UX
* 💾 Data stored using **LocalStorage**
* ✅ Basic validation (e.g., password matching)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Animations)
* JavaScript (Vanilla JS)
